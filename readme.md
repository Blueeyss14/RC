# RC

This is an RC car project using WiFi-based control and live camera streaming. The system uses ESP8266 as the main controller and ESP32-CAM for video feed. Flutter is used as the mobile controller app.

## Components
- ESP8266
- ESP32-CAM
- L298N motor driver
- DC motors
- DC-DC Step Down LM2596
- Servo
- Buzzer

## Features
- WiFi remote control via Flutter app
- Live camera streaming
- Forward / backward / left / right motor control
- Alarm
